## Contexto del proyecto

Este proyecto analiza la base de datos financiera y operativa del negocio de renta de inflables y toro mecánico "Diversiones Cometa".

La base de datos registra información de eventos como:
- ingresos por servicio
- gastos operativos
- salarios del personal
- tipo de inflable rentado
- paquetes de toro mecánico contratados

El objetivo del análisis es comprender el desempeño financiero del negocio y detectar patrones de rentabilidad según el tipo de servicio, temporada y costos operativos.

## Análisis realizado

Se desarrolló un dashboard interactivo que permite explorar el desempeño del negocio según el año y mes seleccionados.

El dashboard muestra indicadores clave como:

- Utilidad bruta anual
- Total de ingresos
- Total de egresos
- Inflables más rentados por mes
- Paquetes de toro mecánico más solicitados
- Distribución de gastos
- Salarios pagados al personal

Esto permite identificar qué servicios generan mayor rentabilidad y en qué áreas se concentran los gastos del negocio.

## Conclusiones principales

- El servicio de toro mecánico combinado con DJ genera los mayores ingresos promedio por evento.
- Los inflables infantiles representan la mayor frecuencia de renta mensual.
- Los gastos más altos corresponden a salarios y mantenimiento del equipo.
- Se observa mayor demanda durante temporadas festivas y vacaciones.
